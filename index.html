🍎 iPhone / iOS Users

SmartShop works on iPhone through Safari and can be added to your Home Screen so it feels like a real app.

  1. Tap Open SmartShop Web App above.
  2. Make sure it opens in Safari.
  3. Tap the Share icon at the bottom of Safari.
  4. Scroll down and tap Add to Home Screen.
  5. Tap Add in the top corner.
  6. Open SmartShop from your Home Screen anytime.
Best tip: if the link opens inside Facebook or another app, tap Open in Safari first, then add it to your Home Screen.

🤖 Android Users

You can open the web app or install the Android APK if you want the downloadable version on your phone.

  1. Tap Download Android APK.
  2. Open the downloaded APK file.
  3. Allow install permissions if your phone asks.
  4. Finish install and open SmartShop.
  5. Log in and start shopping smarter.
Already installed an older version? Download the new APK and install it again to update it.
