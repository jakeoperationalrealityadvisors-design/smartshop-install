SmartShop

A smarter grocery planning tool designed to help reduce wasted trips, save time, and organize shopping more efficiently.

iPhone / iPad Users

SmartShop works great directly in Safari and can be added to your home screen for an app-like experience.

  1. Open SmartShop using the button above
  2. Open the page in Safari
  3. Tap the Share button
  4. Select "Add to Home Screen"
  5. Tap Add

Android Users

You can use SmartShop directly in your browser or install the Android version.

  1. Open SmartShop
  2. Log into your account
  3. Tap Download Android APK
  4. Install the downloaded file
  5. Sign in and start using SmartShop
